Lista que almacene objetos de tipo Person

Crear una aplicacion que gestione una lista ordenada de objetos person. Los objetos person tendran las siguientes propiedades

  • Propiedad: name. Contendra el nombre
  • Propiedad: surname. Contendra el apellido
  • Se necesita que los objetos esten ordenados dentro de la lista
  • Se debe realizar la gestion de errores mediante objetos
  • El codigo usado debe ser reutilizable
  • Debe ser lo mas eficiente que sea posible
